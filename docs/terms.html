利用規約

最終更新日：2025年1月28日

本利用規約（以下「本規約」）は、TextCast（以下「本アプリ」）の利用条件を定めるものです。ユーザーの皆様（以下「ユーザー」）は、本アプリを利用することにより、本規約に同意したものとみなされます。

第1条（適用）

本規約は、本アプリの提供者（以下「当方」）とユーザーとの間の、本アプリの利用に関する一切の関係に適用されます。

第2条（サービス内容）

本アプリは、以下の機能を提供します：

  • テキストの音声読み上げ機能
  • 多様なファイル形式（PDF、Word、画像等）からのテキスト抽出と読み上げ
  • URL からのコンテンツ取得と読み上げ
  • プレイリスト管理機能
  • バックグラウンド再生機能
  • 音声設定のカスタマイズ（速度、ピッチ等）
  • クラウドバックアップ機能（オプション）

第3条（アカウント）

本アプリは、アカウント登録なしで利用できます。ただし、クラウドバックアップ機能を使用する場合は、Apple ID または Google アカウントが必要です。

第4条（プレミアムプラン）

4.1 プレミアムプランの内容

ユーザーは、有料のプレミアムプラン（月額 ¥480）に登録することで、以下の機能を利用できます：

  • プレイリスト数の制限解除（無制限）
  • 広告の完全非表示
  • ファイル処理の制限解除
  • クラウドバックアップの容量無制限
  • 高度な音声設定

4.2 自動更新とキャンセル

  • 自動更新: プレミアムプランは、キャンセルするまで毎月自動的に更新されます
  • 課金タイミング: 登録後、すぐに初回の課金が行われます。以降、毎月同じ日に自動更新されます
  • キャンセル方法:
    • iOS: 設定 → Apple ID → サブスクリプション → TextCast → サブスクリプションをキャンセル
    • Android: Google Play ストア → アカウント → お支払いと定期購入 → 定期購入 → TextCast → 定期購入を解約
  • 返金: キャンセル後も、現在の請求期間の終了日までプレミアム機能を利用できます。日割り返金は行いません
  • キャンセル期限: 次回の更新日の少なくとも24時間前までにキャンセルする必要があります

4.3 無料トライアル（将来提供予定）

当方が無料トライアルを提供する場合、トライアル期間終了後、自動的にプレミアムプランに移行し、課金が開始されます。トライアル期間中にキャンセルした場合、課金は発生しません。
